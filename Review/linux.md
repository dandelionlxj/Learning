## 一、处理目录常用的命令

ls：列出目录

cd：切换目录

pwd：查看当前所在路径

mkdir：创建一个新的目录

rmdir：删除一个空的目录，如果要删除非空目录，则用rm -rf

touch:创建空文件

cp：复制文件或目录 （cp[]

rm：移除文件或目录

mv：移动文件或目录，或修改文件与目录的名称

\>指令：输出重定向，会将原来文件的内容覆盖

\>>指令：追加到文件内容的末尾，不覆盖

echo:输出内容到控制台

## 三、Linux磁盘管理

常用三个命令为 df、duhe fdisk

* df：列出文件系统的整体磁盘使用量
* du：检查磁盘空间使用量，与df命令不同的是Linux du命令是对文件和目录磁盘使用的空间的查看，
* fdisk：用于磁盘分区

**磁盘格式化：**

mkfs[-t  文件系统格式] 装置文件名



**磁盘检验：**

fsck（file system check）用来检查和维护不一致的文件系统。

若系统掉电或磁盘发生问题，可利用fsck命令对文件系统进行检查。

语法：

```
fsck [-t 文件系统] [-ACay] 装置名称
```



**磁盘挂载与卸除**

Linux 的磁盘挂载使用 `mount` 命令，卸载使用 `umount` 命令。

磁盘挂载语法：

```
mount [-t 文件系统] [-L Label名] [-o 额外选项] [-n]  装置文件名  挂载点
```

## 七、压缩和解压缩命令

1、gzip  文件：压缩文件，只能压缩为 *.gz文件

2、gunzip 文件.gz：解压缩

3、zip用于压缩文件  zip [选项] XXX.zip 将要压缩的内容

-  -r :递归压缩，即压缩目录

4、unzip：解压缩文件

-  -d<目录>：指定解压后的存放目录

5、tar：打包指令，最后打包后的文件是.tar.gz的文件 tar[选项] XXX.tar.gz

- -c:产生.tar打包文件

- -v：显示详细信息

- -f：指定压缩后的文件名
- -z：打包同时压缩
- -x：解包.tar 文件

示例：将a1.txt 和a2.txt打包并压缩成a.tar.gz（tar -zcvf a.tar.gz a1.txt a2.txt)

​			将a.tar.gz解压（tar -zxvf a.tar.gz)

​			将a.tar.gz解压到指定目录（tar -zxvf a.tar.gz -C /opt/)



## 八、组管理

1、ls -ahl:查看文件的所有者

2、chown 用户名 文件名：修改文件所有者

3、chgrp 组名 文件名：修改文件所在组

4、usermod -g 组名 用户名：改变用户所在组

5、usermod -d 目录名 用户名：改变用户登录的初始目录



文件权限：

- 第0位：表示文件的类型，-为普通文件，d为目录，l为软连接，c为字符设备（键盘，鼠标），b为块文件（硬盘）
- 第1~3位：表示文件所有者拥有的权限，r为读，w为写，x为执行
- 4~6位：文件所在组的用户的权限
- 7~9位：文件的其他组的用户的权限
- 10位的数字：如果是文件，数字就表示硬链接的数，如果是目录数字就表示该目录的子目录个数
- 文件拥有者
- 所在组
- 文件的大小



## 进程管理

1、ps：查看目前系统中，有哪些正在执行，以及它们执行的情况

- -a：显示当前终端的所有进程信息
- -u：以用户的格式显示进程信息
- -x：显示后台进程运行的参数
- -ef：看父进程

2、kill  [选项]  进程号：通过进程号杀死进程

- -9：强制进程立即停止

3、killall 进程名称：通过进程名称杀死进程，也支持通配符， 

4、pstree [选项]:查看进程树，可以更加直观的来看进程信息

- -p:显示进程的PID
- -u:显示进程的所属用户

5、top：与ps命令很相似，不同之处在于，top在执行一段时间可以更新正在运行的进程（动态监控进程）

- -d 秒数：top命令每隔多少秒更新，默认三秒
- -i：使top不显示任何闲置或者僵尸进程
- -p：通过指定监控进程id来仅仅监控某个进程的状态

6、netstat：查看系统网络情况（netstat -anp)
